How to Use QuHub Trading Bot

Step 1: Install Required Software

If you're using a mobile device without a laptop, install the Trimes app. Otherwise, install Python and Git on your computer.

Step 2: Clone the Repository

Navigate to your desired folder and clone the repository using Git:

git clone https://github.com/tradehub/quotex-bot.git

Step 3: Run the Installation

Navigate to the cloned repository folder and run the installation command:

python install.py

Step 4: Generate API Key

The installation will generate an API key. Copy this key and save it securely.

Step 5: Login to Quotex

Enter your Quotex account credentials. If you have 2FA enabled, you'll need to enter the code when prompted.

Step 6: Configure Trading Parameters

On the trading page, select your account type (demo/real), choose a strategy, and set your trade amount.
